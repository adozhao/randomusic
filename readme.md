### 介绍
目前支持的功能：    
- [X] 支持歌曲的播放控制
- [X] 支持更换歌曲的类别
- [X] 支持最小化隐藏到托盘
- [ ] 支持快捷键切换歌曲
- [ ] 支持快捷键显示和隐藏播放器  
...
### 运行
```
npm i
npm run serve
npm run electron:build
```
