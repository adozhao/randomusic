### 介绍
electron构建的音乐播放器，适合追求简单随心听~  
- 支持歌曲的播放/暂停/切换
- 支持更换歌曲的类别
- 支持最小化隐藏到托盘
### 运行
```
npm i
npm run serve

npm run electron:build #打包为exe文件
```
